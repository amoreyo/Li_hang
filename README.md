## 感知机

初步实现了感知机的原始形式

对偶形式先鸽一鸽

12/25 写完了对偶形式，发现原始形式和对偶形式本质是一模一样的

<code>w = w + LR * y * x 

b  = b + LR * y</code>

那就从算法本身考虑，因为对偶形式提前计算了Gram矩阵，所以在运算判断肯定会比原始形式要来的快，估计对偶形式名字的来源就是Gram是一个对称矩阵吧，再者，实际上运算还能减少，把y也对应乘到Gram中，虽然Gram不对称了，但是计算量进一步的减少了。

当然，这里减少计算量的前提是拿出一部分空间去存储不变的东西。这种形式的优化应该到处都是。
